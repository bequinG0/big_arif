**Собственная реализация длинной арифметики на C++** для работы с произвольно большими целыми числами. Проект разработан в 10 классе как учебная реализация.

## 🔥 Особенности

- Поддержка всех основных арифметических операций (`+`, `-`, `*`, `/`)
- Перегрузка операторов для удобного использования (`++`, `+=`, `==` и др.)
- Работа с отрицательными числами
- Ввод/вывод через стандартные потоки (`cin`/`cout`)
- Векторное хранение цифр числа# big_arif
библиотека для использования больших чисел в С++
